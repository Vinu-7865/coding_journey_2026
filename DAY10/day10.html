EX 01

Reflow vs Repaint

transform opacity GPU composite paint flashing
❌ BAD — reflow
width

Animates width — triggers full layout recalc every frame

✅ GOOD — composite
transform

Animates transform — handled by GPU, zero layout cost

/* BAD — causes reflow every frame */ .box:hover { width: 200px; } /* GOOD — GPU composite only */ .box:hover { transform: scaleX(2); } /* Rule: only animate transform and opacity */

EX 05

CSS Variables + Dark Mode

:root var(--clr-*) data-theme cascade
CSS Variables

Every colour uses var(--clr-*)

Toggle dark mode from the nav bar. One data-theme attribute on <html> switches every colour on the page — no extra rules needed.

:root { --clr-bg: #f8f7f4; --clr-text: #1a1917; } [data-theme="dark"] { --clr-bg: #111110; --clr-text: #e8e5de; } body { background: var(--clr-bg); color: var(--clr-text); } /* one JS line switches the entire page */ document.documentElement.dataset.theme = 'dark';
