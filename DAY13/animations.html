Tailwind Animation Cheat Sheet
Built-in animations
animate-spin    → rotate 360deg infinite
animate-pulse   → opacity pulse infinite
animate-bounce  → bounce up/down infinite
animate-ping    → scale + fade infinite
Transition classes
transition-all       duration-200
hover:-translate-y-1  hover:shadow-lg
hover:scale-105      active:scale-95
focus:ring-2         focus:ring-blue-500/20
Transform classes
translate-y-0    -translate-y-1
translate-x-0    -translate-x-full
scale-100        scale-95 scale-105
rotate-0         rotate-180
Shadow + opacity
shadow-sm  shadow-md  shadow-lg
shadow-blue-500/30  ← colour + opacity
opacity-0  opacity-50  opacity-100
hover:opacity-80
